新手的折腾之路

刚刚地震了，总怕一天被活埋 –好吧，题外话。

社交图标

社交图标的链接，帮助文档说：

去掉设置前的 # 再填写链接即可

这个很简单嘛，看我的。
原github设置：

#GitHub: "#"

修改为：

GitHub: "github.com/CauNong"

嘿嘿， 我真聪明！ 开始部署，神马！ 居然不是想象中的样子？？？？

转了一圈，终于搞定：

GitHub: "//github.com/CauNong"

这才是正确的写法,唉，傻白甜我只剩傻了！
